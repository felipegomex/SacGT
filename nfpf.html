Remessa para Conserto ou Troca de Produto

Pessoa Física

1. Identificar o CPF do Cliente via declaração de conteúdo ou NF.

Caso não tenha:

1.1. Pesquisar via número autorização de postagem.

2. Abrir O.S com dados do cliente.

→ Abrir O.S

2.1. Caso tenha cadastro no sistema, abrir O.S e preencher na planilha com o CL.

2.2. Caso não tenha cadastro no sistema, não abrir O.S e preencher na planilha como "sem cadastro".

3. Verificar Produto

3.1. Identificar e listar o problema.

3.2. Verificar se o produto tem conserto com os técnicos.

3.2. Caso não tenha, recolher código do produto e solicitar um novo via estoque.

4. Gerar Etiqueta de Postagem

→ Portal Postal

4.1. Preencher com os dados do cliente e a declaração de conteúdo.

4.2. Caso cliente tenha enviado NF de compra, reenviar de volta com o produto.

4.3. Recolher código de rastreio e preencher na planilha.

4.4. Imprimir a lista de postagem.

5. Embalar Produto

5.1. Não deixar nenhuma logo fora da embalagem.

5.2. Colar etiquetas do Correios.

5.3. Caso tenha sido feito a troca, entregar produto para o Léo na caixa. (retirar informações).

5.4. Entegar produto para recepção com a lista de postagem respectiva.
